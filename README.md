# 医院信息管理系统 python1276

## 项目概述

医院信息管理系统基于 Django 框架开发，采用 Python 语言编写。系统旨在提高医院内部信息管理的效率，通过科学的流程设计，实现数据的有效整合与管理。

## 技术栈

- **后端框架**：Django
- **编程语言**：Python 3.7/3.8
- **数据库**：MySQL 5.7 或更高版本
- **数据库工具**：Navicat 11（推荐）
- **开发环境**：PyCharm、VS Code
- **前端框架**：Vue.js

## 功能模块

系统主要包括以下功能模块：

- 患者信息管理
- 医生排班与预约
- 药品库存管理
- 费用结算
- 系统权限与用户管理

## 系统角色

- 系统管理员
- 医生
- 药剂师
- 挂号员

## 运行环境

- 操作系统：Linux 或 Windows
- Python 环境要求：3.7 或 3.8 版本
- 数据库：MySQL 5.7 或更高版本

## 部署步骤

1. 环境准备：安装 Python、MySQL、Navicat
2. 克隆源码到本地
3. 安装项目依赖：通过 pip 安装 requirements.txt 文件中列出的依赖
4. 数据库配置：创建数据库，执行数据库迁移命令以生成数据表
5. 运行服务器：使用 manage.py 文件启动开发服务器
